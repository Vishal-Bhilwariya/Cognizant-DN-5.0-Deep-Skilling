## 🔍 How to Access Weekly Content

### On GitHub

1. Open this repository.
2. Click the **Branch Selector** dropdown.
3. Select the required week's branch.
4. Browse the hands-on practice files and solutions.

📌 Each branch contains the complete practice work for that specific week.
